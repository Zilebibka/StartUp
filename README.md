

## 1.  Telegram‑бот для автоматического создания постов

- **Идея**  
  Автоматизация ручного создания постов для Telegram‑каналов.

- **Решение**  
  Бот самостоятельно парсит новости с указанных пользователем сайтов или Telegram‑каналов.  
  С заданным интервалом проверяет наличие новых материалов и при обнаружении с помощью ИИ генерирует готовый к публикации пост, отправляя его пользователю.

- **Анализ конкурентов**  
  На данный момент прямые конкуренты не найдены.

- **Финансы**  
  - **Заработок:** подписка или единоразовый платёж за использование бота (без ограничений по тематике каналов). **В будущем:** реклама бота.
  - **Личные затраты:** аренда серверов.

## 2.  Музыкальный плеер в Telegram (MiniApp)

- **Идея**  
  Прослушивание музыки непосредственно в Telegram.

- **Решение**  
  MiniApp‑плеер с возможностью:
  - просматривать свои плейлисты;
  - сортировать и искать треки;
  - добавлять музыку в собственные подборки.

- **Анализ конкурентов**  
  Прямых конкурентов с аналогичным функционалом нет.  
  Косвенные — Telegram‑боты для поиска музыки по запросу, но без функций сортировки и группировки треков.

- **Финансы**  
  - **Заработок:** подписка на срок и/или реклама в бесплатной версии.
  - **Личные затраты:** серверы для MiniApp. **В будущем:** продвижение приложения.

## 3.  Единое приложение для МИРЭА

- **Идея**  
  Отсутствие единого структурированного приложения — сервисы университета разбросаны:  
  расписание, СДО, ЛКС, студенческий билет, электронная зачётка.

- **Решение**  
  Нативные приложения для Android и iOS, объединяющие:
  -  расписание;
  -  СДО (задания, видеолекции, материалы);
  -  личный кабинет студента (задолженности, оплата обучения и т.д.);
  -  студенческий билет с QR‑кодом;
  -  электронную зачётную книжку.

- **Анализ конкурентов**  
  Прямых конкурентов нет. Косвенные — отдельные сайты и приложения, не образующие целостную экосистему.

- **Финансы**  
  - **Заработок:** реклама.

## 4.  Платформа для поиска поставщиков

- **Идея**  
  Ручной поиск поставщиков среди ненадёжных и малоизвестных источников.

- **Решение**  
  Веб‑платформа / сайт, где:
  - поставщики оставляют заявки и заполняют информацию о себе;
  - пользователи ищут нужных поставщиков по ключевым словам или через поиск.

- **Анализ конкурентов**  
  Прямые конкуренты не обнаружены.  
  Косвенные — множество разрозненных сайтов отдельных поставщиков.

- **Финансы**  
  - **Личные затраты:** сервер, реклама.  
  - **Заработок:** реклама на платформе.
