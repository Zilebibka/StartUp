## 4. Маркетплейс «мёртвого кода» — покупка и продажа заброшенных проектов (WebApp) (Денис)
**Идея**
- Разработчики бросают тысячи проектов на полпути. Платформа, где можно продать свой недоделанный проект (с кодом, документацией, доменом, пользовательской базой) другому разработчику, который его доделает и   монетизирует.

**Решение**
- Веб-платформа с возможностью:

*листинга заброшенных проектов с автоматическим аудитом: качество кода, стек технологий, оставшийся технический долг, потенциальная аудитория;
AI-оценки стоимости проекта на основе кодовой базы, трафика, домена и конкурентного анализа;
безопасной сделки через эскроу: покупатель получает код и доступы только после подтверждения, продавец — деньги.*

**Анализ конкурентов**
- Прямых конкурентов с AI-аудитом кода и оценкой заброшенных проектов нет.
Косвенные — Flippa (продажа работающих сайтов, не кода), MicroAcquire (только SaaS с выручкой), GitHub (бесплатные open-source, не маркетплейс).*

**Финансы**

- Заработок: комиссия 10–15% с каждой сделки, платный расширенный AI-аудит, премиум-листинг с повышенной видимостью.
- Личные затраты: разработка платформы и AI-аудитора кода. В будущем: создание инкубатора для «воскрешённых» проектов, интеграция с GitHub/GitLab.
